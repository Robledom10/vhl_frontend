Políticas de cancelación

  • Cancelación gratuita hasta 5 días antes del viaje.
  • 50% de reembolso si cancela 48 horas antes.
  • No hay devolución el mismo día del viaje.
  • Se permite cambio de fecha una vez.

Requisitos

  • Cédula de ciudadanía obligatoria.

  • Edad mínima: 5 años.

  • Seguro obligatorio.
